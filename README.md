# The-Haas-University

web development project for The Haas University created as part of Saylani class assignments.

## Features
- Simple and clean web design  
- Basic navigation menu for easy browsing  
- Partially responsive layout (works on some devices)  
- Sections for Home, About, Courses, and Contact  

## Installation
1. Clone the repository: `git clone <repository-url>`  
2. Open the project folder  
3. Open `index.html` in your browser or run with a local server
